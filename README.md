# Defesa Cognitiva

Aplicação web simples para analisar texto e sinalizar **padrões de manipulação emocional**.

## Como usar

1. Abre `index.html` no navegador.
2. Cola uma mensagem suspeita.
3. Clica em **Analisar manipulação**.
4. Vê o score, nível de risco e técnicas detectadas.

## Nota importante

A app **não faz fact-checking** e **não decide o que é verdade**.
Ela só avalia estrutura de persuasão para ajudar a desacelerar reações automáticas.
